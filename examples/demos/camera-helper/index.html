Use the + button on the bottom-left to record the camera position. Move around and add a few points, and then hit play or use the slider to scrub between them.

The JSON export contains individual points as well as data for an Animation clip. It can be used for Story Point controls, Scroll controls and Path Point controls.

You can export a draft JSON to keep working on the points later. Once you're finished you can export a smaller production copy.

Use arrow/WASD keys or mouse wheel/trackpad to move around, and click and drag to turn/rotate the camera. U/N keys raise/lowers the camera.

Note: This isn't intended to be used on mobile.
